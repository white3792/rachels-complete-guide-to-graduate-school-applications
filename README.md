# Rachel's Complete Guide to Graduate School Applications

A complete guide to applying to Physics PhD programs, created by an Ohio State University astrophysics graduate accepted to seven programs and awarded the NSF-GRFP. Includes timelines, email templates, writing advice, and downloadable application materials.

🌐 **[View the full guide](https://white3792.github.io/rachels-complete-guide-to-graduate-school-applications/)**
